# 版本记录

- V1.0
  - 自定义文字支持多行输入（`dark_asktext` 多行文本对话框）
  - 样式面板新增描述文字字体选择和粗体开关
  - 描述文字可独立设置字体和粗细（`text_font_family`/`text_bold`）
  - 新增 `dark_asktext` 深色主题多行文本输入对话框
  - 状态持久化新增 `text_font_family` 和 `text_bold` 字段
  - 帮助文本和对话框提示适配新功能
  - 版本号跳至 V1.0

- V0.9
  - 全屏按钮在进入全屏后变为"退出全屏"，点击退出全屏
  - Escape 键只在全屏状态退出全屏，非全屏时不触发
  - 初始化 `_toolbar_anim_id` 避免隐式依赖
  - `load_user_state` 加载失败时打印警告信息，不静默吞异常
  - 国际化 filedialog 和 colorchooser 的窗口标题
  - 样式面板关闭时清理悬空引用（`_style_swatch` / `_font_var` 等）
  - 窗口 resize 增加 100ms 防抖，降低拖拽时背景重绘频率
  - `format_hms` 上限锁定 99:59:59，防止小时溢出布局

- V0.8
  - 全面视觉重塑：现代深色主题（#080811 底色），新色板贯穿全部 UI
  - 时间样式自定义：样式面板新增字体选择（9 种）、粗体开关、投影开关
  - 时间显示优化：移除粗体、增大默认字号至 120、添加柔和投影
  - 描述文字与版本信息独立配色（#9494b8 / #5a5a7a），与主时钟色分离
  - 功能栏重设计：更精致的磨砂玻璃模拟（双层 Frame 结构）、紧凑按钮
  - 功能栏动画：隐藏/显示时 cubic-ease 滑动过渡（~160ms）
  - 自定义深色对话框：替换所有系统原生 dialog/messagebox，统一深色主题
    - dark_messagebox：支持 info/error 双模式，顶部彩色 accent 条
    - dark_askstring：深色输入框，焦点高亮边框
    - dark_askinteger：带实时校验与行内错误提示
    - 全部对话框居中于主窗口、支持 Enter/Esc 快捷键
  - 对话框文案完整双语适配（中/英），废弃硬编码中文字符串
  - 样式面板统一深色主题、输入框与滑块配色匹配
  - 新增状态持久化：字体、粗体、投影设置保存至 user_state.json
  - 修复语言切换卡顿：on_resize 增加尺寸守卫避免无效背景重载、update_ui 移除冗余 position_elements、mode_label 即时刷新

- V0.7
  - 修复功能栏刷新卡顿：UI 改为”仅在显示内容变化时刷新”，降低无效重绘
  - 模式标签与状态文本按变更更新，减少连续 configure 带来的卡顿

- V0.6
  - 新增“使用记录恢复”功能：自动保存上次使用的背景图片和自定义文本
  - 下次启动应用时自动加载上次图片与文本
  - 重置版本记录：删除旧的 V0.6 之后版本，后续从 V0.6 继续递增

- V0.5
  - 新增中英文界面切换按钮（功能栏可在中文/英文之间切换）
  - 模式显示、帮助文案、样式面板核心文案支持双语切换
  - 新增 GitHub Actions 发布工作流：推送版本 tag 后自动打包 exe 并上传到 GitHub Releases

- V0.4
  - 功能栏改为仿磨砂玻璃样式
  - 模式显示从画布状态区迁移到功能栏内实时显示
  - 新增隐藏功能栏选项；隐藏后仅保留一个箭头按钮用于展开

- V0.3
  - 样式滑块模块新增手动数字输入框（回车或失焦应用）
  - 滑块与输入框双向同步，超范围值自动夹紧到允许区间

- V0.2
  - 封装时显式加入 Pillow 相关模块，确保 exe 内可用背景图处理
  - 新增背景图缩放功能：样式面板可通过滑块调整背景缩放比例(20%-300%)，实时生效

- V0.1
  - 新增统一版本号规则，界面显示版本信息
  - 修复背景图不可用：未安装 Pillow 时支持 png/gif/ppm/pgm，并对不支持格式给出明确提示
