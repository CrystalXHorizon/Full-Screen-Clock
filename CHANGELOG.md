# 版本记录

- V0.8
  - 清理仓库与本地构建产物，不再保留打包中间文件
  - 新增 `.gitignore`，忽略 `build/`、`dist/`、`*.spec`、`__pycache__/`
  - 新增 `scripts/build_client.ps1`：客户端默认输出到 `Downloads` 文件夹

- V0.7
  - Release 工作流支持两种触发：推送 tag 或在 GitHub 发布 Release
  - 自动按版本名生成并上传 exe 资产（DesktopClock-vX.Y.exe）
  - 自动生成 Release Notes

- V0.6
  - 生成可直接运行的集成环境客户端，文件名追加版本号（本机与 Release 资产一致）
  - 发布流程改为按 tag 自动输出带版本号的 exe 文件名

- V0.5
  - 新增中英文界面切换按钮（功能栏可在中文/英文之间切换）
  - 模式显示、帮助文案、样式面板核心文案支持双语切换
  - 新增 GitHub Actions 发布工作流：推送版本 tag 后自动打包 exe 并上传到 GitHub Releases

- V0.4
  - 功能栏改为仿磨砂玻璃样式
  - 模式显示从画布状态区迁移到功能栏内实时显示
  - 新增隐藏功能栏选项；隐藏后仅保留一个箭头按钮用于展开

- V0.3
  - 样式滑块模块新增手动数字输入框（回车或失焦应用）
  - 滑块与输入框双向同步，超范围值自动夹紧到允许区间

- V0.2
  - 封装时显式加入 Pillow 相关模块，确保 exe 内可用背景图处理
  - 新增背景图缩放功能：样式面板可通过滑块调整背景缩放比例(20%-300%)，实时生效

- V0.1
  - 新增统一版本号规则，界面显示版本信息
  - 修复背景图不可用：未安装 Pillow 时支持 png/gif/ppm/pgm，并对不支持格式给出明确提示
